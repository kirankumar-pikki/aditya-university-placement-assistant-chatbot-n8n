# 🎓 Aditya University Placement Assistant

An AI-powered Placement & Career Development Assistant built for Aditya University students.

This project provides structured JSON data and chatbot-ready automation workflows containing placement statistics, recruiter details, internship information, and career development services.

Designed for seamless integration with automation tools like n8n, chatbots, and APIs.

---

## 🚀 Project Overview

The Aditya University Placement Assistant helps students:

- Explore placement opportunities  
- View recruiter information  
- Access internship details  
- Understand placement statistics  
- Get career development guidance  
- Integrate data into chatbots or automation systems  

This repository contains structured JSON datasets that can be used for:

- Chatbot responses
- API integrations
- Automation workflows (n8n)
- Web applications
- Placement dashboards

---

## 📊 Features

- 📌 Structured Placement Data
- 🏢 Top Recruiter Information
- 💼 Internship Opportunities
- 📈 Placement Statistics (Year-wise)
- 📞 Placement Cell Contact Details
- 🤖 Chatbot-ready JSON format
- ⚙️ Automation-friendly design
